Flask NewsBoard

← Back to Home

This project is a local Flask web application that demonstrates a real-world implementation of the Publish–Subscribe (Pub/Sub) communication model. It simulates a simple news board platform where users can subscribe to receive and view posts published by an administrator.

The system provides a clear example of event-driven communication and message distribution in a local environment, showing how subscribers dynamically receive updates from publishers.
