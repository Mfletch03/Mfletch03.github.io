This quiz app allows users to test their knowledge in various subjects.

Features:

  • account creation and login system for personalized quiz experiences.
  • Interactive quizzes with multiple-choice questions.
  • Multiple categories of quizzes using Open Trivia DB API (e.g., Video Games, Books, Sports).
  • Score tracking and feedback at the end of the quiz.
  • A MySQL database to store quiz data and user scores for persistent tracking.

Technologies used: HTML, CSS, JavaScript, MySQL
